# Exercícios de Programação

Este repositório contém soluções para **5 exercícios de programação**, desenvolvidos utilizando **HTML** e **JavaScript**. A linguagem **HTML** foi escolhida para permitir a visualização dos resultados diretamente no navegador, facilitando o acompanhamento do progresso de cada questão.
